HTML

  • HTML: Hypert Text Markup Language
  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • Elements that are nested iside other elements are called child elements.
  • The element that holds all the children is known as the parent element.

CSS

  • CSS: Cascading Style Sheet
  • Language used to describe presentation of a website that's created with HTML.
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.

Git

  • Git: Free open source version control system that allows web developers to revert to previous versions of the code if a problem is discovered.
  • Github: Largest provider of internet hosting for software and web development code. Provides version control functionality of git in a central repository.
  • Git status: checks what branch we are currently on.
  • Git checkout -b branch-name: creates a new branch and switches to it.

Javascript

  • Javascript: Provides logic and interactivity for websites
  • In coding parlance, a variable is simply a named container that allows us to store data in our code
  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script. In JavaScript, code runs from the first statement at the top of the page, down to the last statement at the bottom of the page.
